# Эквалайзер - Краткая инструкция

## 📍 Текущий статус

**Версия:** MVP (UI-only)  
**Прогресс:** ✅ Готов для демонстрации

### Что работает

- ✅ Полнофункциональный UI (10 полос)
- ✅ 8 предустановленных пресетов
- ✅ Включение/выключение
- ✅ Ручная настройка каждой полосы
- ✅ Сброс к плоскому звуку
- ✅ Интеграция с Zustand store
- ✅ Синхронизация с аудио-движком (через заглушки)
- ✅ Автоматическое применение настроек

### Что НЕ работает

- ❌ Реальная обработка звука (запланировано Post-MVP)

## 🎵 Как использовать

### В коде

```typescript
import { useEqualizerStore } from '@/features/equalizer/store/equalizerStore';

function MyComponent() {
  const { 
    enabled,           // Включен ли эквалайзер
    bands,             // Массив значений gain [10]
    currentPreset,     // Текущий пресет
    toggleEqualizer,   // Включить/выключить
    setBandGain,       // Изменить полосу
    applyPreset,       // Применить пресет
    resetToFlat,       // Сброс
  } = useEqualizerStore();

  // Включить эквалайзер
  toggleEqualizer();

  // Применить пресет
  applyPreset('rock');

  // Настроить полосу вручную (индекс 0-9, gain -12 до +12)
  setBandGain(0, 6);
}
```

### В UI

Экран эквалайзера: **Навигация → Equalizer Tab**

1. **Включение/выключение** - переключатель вверху экрана
2. **Пресеты** - горизонтальный скролл с кнопками пресетов
3. **Полосы** - 10 вертикальных слайдеров
4. **Сброс** - кнопка "Reset to Flat" внизу

## 🔧 Технические детали

### Частоты

| Band | Frequency | Gain Range |
|------|-----------|------------|
| 0    | 31 Hz     | -12 to +12 dB |
| 1    | 63 Hz     | -12 to +12 dB |
| 2    | 125 Hz    | -12 to +12 dB |
| 3    | 250 Hz    | -12 to +12 dB |
| 4    | 500 Hz    | -12 to +12 dB |
| 5    | 1 kHz     | -12 to +12 dB |
| 6    | 2 kHz     | -12 to +12 dB |
| 7    | 4 kHz     | -12 to +12 dB |
| 8    | 8 kHz     | -12 to +12 dB |
| 9    | 16 kHz    | -12 to +12 dB |

### Пресеты

1. **Flat** - все полосы на 0 dB
2. **Rock** - усиление басов и высоких
3. **Pop** - средние частоты
4. **Jazz** - сбалансированный
5. **Classical** - естественный звук
6. **Electronic** - глубокий бас + яркие высокие
7. **Hip-Hop** - мощный бас
8. **Vocal Boost** - акцент на вокале (1-4 kHz)

## ⚠️ Важно

**Текущая версия MVP:**
- Эквалайзер работает в UI-only режиме
- Все методы audioPlayer.setEqualizer* - это заглушки
- Настройки сохраняются в памяти (не в storage)
- Реальная обработка звука будет добавлена в Post-MVP

**Логирование:**
В режиме разработки (`__DEV__`) в консоль выводятся:
- Изменения состояния эквалайзера
- Вызовы методов audioPlayer
- Применение пресетов

## 🐛 Известные ограничения

1. Нет реальной обработки звука
2. Настройки не сохраняются между сессиями
3. Нет визуализации частотного отклика
4. Пресеты нельзя редактировать или создавать новые

Эти ограничения будут устранены в Post-MVP версии.
