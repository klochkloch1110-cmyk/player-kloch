# 🚀 Автоматическая сборка APK через GitHub Actions

## 📋 Что это дает?

- ✅ **Автоматическая сборка APK** при каждом push в репозиторий
- ✅ **Без установки Android Studio** на локальной машине
- ✅ **Бесплатно** для публичных репозиториев
- ✅ **Готовый APK** можно скачать из раздела Actions

---

## 🎯 Как это работает

1. Ты делаешь `git push` в репозиторий
2. GitHub Actions автоматически:
   - Устанавливает Node.js, Java JDK 17, Android SDK
   - Устанавливает npm зависимости
   - Собирает APK через Gradle
   - Сохраняет готовый APK как artifact
3. Ты скачиваешь готовый APK из интерфейса GitHub

---

## 📝 Инструкция по первой настройке

### Шаг 1: Проверь структуру репозитория

Убедись, что репозиторий имеет такую структуру:

```
player-kloch/
├── AudioPlayer/          # Проект React Native
│   ├── android/
│   ├── src/
│   ├── package.json
│   └── .github/
│       └── workflows/
│           └── build-android.yml  # Workflow файл
└── README.md
```

### Шаг 2: Закоммить workflow файл

```bash
cd c:/Users/95/Desktop/Диван/AudioPlayer
git add .github/workflows/build-android.yml
git add docs/GITHUB_ACTIONS_BUILD.md
git commit -m "feat: добавлен GitHub Actions workflow для автоматической сборки APK"
git push origin main
```

*Примечание:* Замени `main` на `master` или другую ветку, если используешь другое имя.

### Шаг 3: Проверь статус сборки

1. Перейди на https://github.com/klochkloch1110-cmyk/player-kloch
2. Открой вкладку **Actions**
3. Увидишь запущенную сборку "Build Android APK"
4. Дождись завершения (обычно 5-8 минут)

### Шаг 4: Скачай готовый APK

После успешной сборки:

1. Открой завершенную сборку в разделе Actions
2. Прокрути вниз до раздела **Artifacts**
3. Скачай файл `app-release.zip`
4. Распакуй архив — внутри будет `app-release.apk`

## 📱 Установка APK на телефон

### Способ 1: Через USB

1. Включи **"Режим разработчика"** на Android:
   - Настройки → О телефоне → 7 раз нажми на "Номер сборки"
2. Включи **"Отладка по USB"**:
   - Настройки → Для разработчиков → Отладка по USB
3. Подключи телефон к компьютеру через USB
4. Скопируй `app-release.apk` на телефон
5. Открой файл на телефоне и установи

### Способ 2: Через облако / мессенджер

1. Отправь `app-release.apk` себе через Telegram / WhatsApp / облако
2. Скачай файл на телефон
3. Открой файл и установи
4. Если система блокирует установку — разреши установку из неизвестных источников

## 🔄 Автоматическая сборка при изменениях

Workflow настроен так, что APK будет собираться автоматически:

- ✅ При каждом `git push` в ветки `main`, `master`, `develop`
- ✅ При создании Pull Request
- ✅ Вручную через кнопку "Run workflow" в разделе Actions

## ⚙️ Параметры сборки

### Текущие настройки:

- **Node.js:** v20
- **Java JDK:** 17 (Temurin)
- **Build Type:** Release (оптимизированная сборка)
- **Хранение artifacts:** 30 дней
- **Кэширование:** Gradle, npm modules

### Время сборки:

- Первая сборка: ~8-10 минут (установка всех зависимостей)
- Последующие: ~4-6 минут (благодаря кэшированию)

## 🐛 Решение проблем

### Сборка падает с ошибкой "Gradle build failed"

Проверь:
1. Все зависимости в `package.json` установлены корректно
2. Файл `android/gradle.properties` настроен правильно
3. Нет конфликтов версий в `android/build.gradle`

### APK не появляется в Artifacts

Убедись:
1. Сборка завершилась успешно (зеленая галочка)
2. Путь к APK правильный: `AudioPlayer/android/app/build/outputs/apk/release/app-release.apk`

### Превышен лимит бесплатных минут

Для публичных репозиториев лимита нет.  
Для приватных: 2000 минут/месяц (примерно 250 сборок).
