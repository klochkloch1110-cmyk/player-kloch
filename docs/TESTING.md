# Руководство по запуску и тестированию

## Предварительные требования

- Node.js >= 22.11.0
- npm или yarn
- Android Studio (для Android) или Xcode (для iOS)
- Физическое устройство или эмулятор

## Установка зависимостей

```bash
cd AudioPlayer
npm install --legacy-peer-deps
```

> **Примечание:** Флаг `--legacy-peer-deps` необходим из-за несовместимости React 19 с некоторыми библиотеками.

## Подготовка к запуску

### Android

1. Откройте Android Studio
2. Создайте эмулятор или подключите физическое устройство
3. Убедитесь, что USB debugging включен (для физического устройства)

### iOS (только macOS)

1. Откройте Xcode
2. Установите CocoaPods зависимости:
   ```bash
   cd ios
   pod install
   cd ..
   ```

## Запуск приложения

### Запуск Metro Bundler

В первом терминале:

```bash
npm start
```

### Запуск на Android

Во втором терминале:

```bash
npm run android
```

### Запуск на iOS

```bash
npm run ios
```

## Подготовка тестовых данных

Приложение сканирует следующие директории:

### Android
- `/storage/emulated/0/Music`
- `/storage/emulated/0/Download`

### iOS
- Documents directory
- Main Bundle

**Чтобы добавить музыку:**

1. **Android эмулятор:**
   - Используйте adb для загрузки файлов:
   ```bash
   adb push /path/to/music/file.mp3 /sdcard/Music/
   ```

2. **Android физическое устройство:**
   - Скопируйте файлы в папку Music через USB или облачное хранилище

3. **iOS симулятор:**
   - Добавьте файлы в Bundle проекта через Xcode

4. **iOS физическое устройство:**
   - Используйте iTunes или Finder для синхронизации

## Тестирование функционала

### 1. Проверка разрешений

При первом запуске:
- ✅ Приложение должно запросить доступ к медиатеке

1. Нажмите на трек в списке

   - ✅ Play/Pause работает
   - ✅ Next/Previous переключают треки
   - ✅ Прогресс-бар обновляется в реальном времени

### 4. Полноэкранный плеер

1. Нажмите на мини-плеер или "Now Playing" в навигации
   - ✅ Откроется полноэкранный плеер
   - ✅ Отображается обложка (или placeholder)
   - ✅ Название трека и исполнитель
   - ✅ Прогресс-бар с временем
   - ✅ Элементы управления

2. Проверьте функции:
   - ✅ Перемотка через прогресс-бар
   - ✅ Shuffle (иконка должна менять цвет)
   - ✅ Repeat (off → one → all → off)

### 5. Эквалайзер

1. Откройте экран эквалайзера

   - ✅ Кнопка Reset сбрасывает на Flat

### 6. Фоновое воспроизведение

1. Сверните приложение (Home button)
   - ✅ Музыка продолжает играть
   - ✅ Уведомление с элементами управления
   - ✅ Кнопки в уведомлении работают

## Известные проблемы и их решение

### Проблема: "No tracks found"

**Решение:**
1. Убедитесь, что музыкальные файлы находятся в правильной директории
2. Проверьте формат файлов (mp3, flac, aac, wav, ogg, m4a)
3. Попробуйте кнопку "Повторить сканирование"

### Проблема: "Permission denied"

**Решение:**
1. Переустановите приложение
2. Вручную предоставьте разрешение в настройках устройства:
   - Android: Settings → Apps → AudioPlayer → Permissions → Storage
   - iOS: Settings → Privacy → Media & Apple Music → AudioPlayer

### Проблема: Аудио не воспроизводится

**Решение:**
1. Проверьте логи Metro bundler на ошибки
2. Убедитесь, что файл существует и доступен
3. Попробуйте другой трек

### Проблема: Metro bundler ошибка

**Решение:**
```bash
# Очистить кеш
npm start -- --reset-cache

# Или
npx react-native start --reset-cache
```

### Проблема: Build failed

**Решение:**

Android:
```bash
cd android
./gradlew clean
cd ..
npm run android
```

iOS:
```bash
cd ios
rm -rf Pods Podfile.lock
pod install
cd ..
npm run ios
```
