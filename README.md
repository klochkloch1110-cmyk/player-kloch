# 🎵 Audio Player - Мобильный аудиоплеер

Современный мобильный аудиоплеер с качественным воспроизведением, 10-полосным эквалайзером и неоморфным дизайном.

## ✨ Особенности

- 🎨 **Неоморфный дизайн** - современный UI с мягкими тенями и плавными переходами
- 🎚️ **10-полосный эквалайзер** - профессиональная настройка звука (31Hz - 16kHz)
- 🎵 **Высококачественное воспроизведение** - поддержка форматов MP3, FLAC, AAC, WAV, OGG
- 🔄 **Режимы повтора** - один трек, все треки, без повтора
- 🔀 **Shuffle** - случайное воспроизведение
- 📚 **Библиотека** - организация по трекам, альбомам, исполнителям
- 🎼 **Плейлисты** - создание и управление плейлистами
- 🌓 **Темы** - светлая и темная тема

## 🛠️ Технологический стек

- **React Native 0.85** - кроссплатформенная разработка
- **TypeScript** - типизация
- **Zustand** - управление состоянием
- **React Navigation 6** - навигация
- **React Native Reanimated** - производительные анимации
- **FlashList** - оптимизированные списки
- **React Native Track Player** - аудио-движок

## 📦 Установка

### Требования

- Node.js >= 22.11.0
- npm или yarn
- React Native CLI
- Android Studio (для Android) или Xcode (для iOS)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd AudioPlayer
```

2. Установите зависимости:
```bash
npm install
```

3. Для iOS дополнительно:
```bash
cd ios && pod install && cd ..
```

## 🚀 Запуск

**Без установки Android Studio! Получи готовый APK за 5 минут.**

1. Запушь код в GitHub

3. Скачай готовый APK из Artifacts

📖 Подробная инструкция: [QUICK_BUILD_APK.md](./docs/QUICK_BUILD_APK.md)

### Вариант 2: Локальная сборка

#### Android

```bash

```

#### iOS

```bash
npm run ios
```

```bash

```

## 🎨 Дизайн-система

### Неоморфизм

Приложение использует неоморфный дизайн с характерными особенностями:

- **Двойные тени**: светлая (сверху-слева) и темная (снизу-справа)
- **Мягкое размытие**: blur 20px
- **Состояния**: flat, pressed (concave), convex
- **Монохромная палитра** с акцентным цветом

### Цветовая схема

#### Светлая тема
- Background: `#E0E5EC`
- Primary: `#6C63FF`
- Text: `#2E384D`

#### Темная тема
- Background: `#2E3440`
- Primary: `#88C0D0`
- Text: `#ECEFF4`

| 5      | 500 Hz  |

| 7      | 2 kHz   |
| 8      | 4 kHz   |
